**Getting Started**
* Follow these steps to set up the dataset and run the project notebooks in Google Colab.
   1. Download the ECONet.zip file and unzip the file.
      * Windows and Macs: Double-click the .zip file to extract it
      * Linux / terminal: Run unzip ECONet.zip
   2. Upload the files to Google Drive.
      * Go to Google Drive and navigate to My Drive.
      * Create a new folder named exactly ECONet or the file paths in the notebook will not resolve correctly.
      * Upload all extracted files into the ECONet folder. It should contain:
         - train.csv
         - test.csv
         - ECONET_QC_Readme.xlsx
         - ECONET.md
         - full/ (folder containing per-station CSVs)
   3. Run the Notebook in Google Colab.
      * Open the project .ipynb file in Google Colab.
      * Run the first cell to mount your Google Drive. A prompt will ask you to authorize Colab to access your Drive — follow the instructions to complete authorization.
      * Once the Drive is mounted, run the remaining cells in order. The notebook will look for your data at: /content/drive/MyDrive/ECONet/
